# Medixflow - AZP Afsprakensysteem

Medixflow is een webapplicatie ontworpen om het proces van het maken van medische afspraken bij het Academisch Ziekenhuis Paramaribo te digitaliseren en te versnellen.

## 🔍 Over het Project
Dit project is ontwikkeld als onderdeel van semester 3 blok 2 van het UNASAT studierichting Software Engineering. Het doel is om een gebruiksvriendelijke interface te bieden voor zowel patiënten als artsen/administratie.

## 🛠️ Gebruikte Technologieën
- HTML5
- CSS3 (Custom Variables & Flexbox)
- JavaScript (Vanilla JS)
